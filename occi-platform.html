OCCI Platform

The OCCI Platform specification defines kinds and extensions relating to management of cloud platform services (PaaS).

Table 1. Common Attributes

AttributeTypeDescription
occi.platform.versionFloat (e.g. 1.0)Version of the OCCI Platform specification

Kinds

Cloud platform services can be modeled using the following primary kinds:

Table 2. Kinds

KindURIDescription
queuehttp://purl.org/occi/kind/queueAn ordered list of tasks or messages
databasehttp://purl.org/occi/kind/databaseAn organised body of related information

Queue

A queue manages an ordered list of tasks or messages for transmission and/or processing.

Table 3. Queue Attributes

AttributeTypeDescription
occi.queue.policyEnum (fifo, filo)Status of the compute resource

Database

A database manages an organised body of related information.

Table 4. Database Attributes

AttributeTypeDescription
occi.database.sizeFloat (10^9 bytes)Database size in gigabytes (e.g. 10, 0.5)

Extensions

Various extensions provide for more advanced management functionality such as billing, monitoring and reporting.
