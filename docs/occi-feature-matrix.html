Cloud API feature Matrix

Existing APIs for IaaS based Clouds already fulfill some of the Use cases and also address some requirements. The following list of APIs have been evaluated during the creation of the OCCI working group process.

  • OCCI - Open Cloud Computing Interface

  • EC2 - Amazon Elastic Compute Cloud

  • EH - ElasticHosts

  • FS - Flexiscale

  • GG - GoGrid

  • SC - Sun Cloud API

  • CS - Rackspace Cloud Servers

  • VM - WMware vSphere

While this is not a complete or in depth evaultation of the APIs the following Matrix can used as a basic reference point for API comparism.
