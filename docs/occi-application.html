OCCI Application

The OCCI Application specification defines kinds and extensions relating to management of cloud application services (SaaS).

Table 1. Common Attributes

AttributeTypeDescription
occi.application.versionFloat (e.g. 1.0)Version of the OCCI Application specification

Kinds

Cloud application services can be modeled using the following primary kinds:

Table 2. Kinds

KindURIDescription
contacthttp://purl.org/occi/kind/contactA contact; a person, a venue such as a club or a restaurant, or an organisation.
documenthttp://purl.org/occi/kind/documentA word processing document
eventhttp://purl.org/occi/kind/eventA calendar event
presentationhttp://purl.org/occi/kind/presentationA visual representation
spreadsheethttp://purl.org/occi/kind/spreadsheetA worksheet

Contact

A contact is a person, venue, organisation, etc.

Extensions

Various extensions provide for more advanced management functionality such as billing, monitoring and reporting.
